# UI Guidelines

Core UI guidelines for the TODO app:

- Use Material components for all primary UI elements, including layout containers, form controls, dialogs, and navigation.
- Use a consistent color palette with clearly defined primary, secondary, success, warning, and error colors.
- Use consistent button styles by intent (primary, secondary, and destructive), with clear states for hover, focus, disabled, and loading.
- Meet accessibility requirements, including keyboard navigation, visible focus indicators, color contrast that meets WCAG AA, and semantic labels for interactive controls.
